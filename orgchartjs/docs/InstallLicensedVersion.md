# Licensed version installation

This document shows you how to install the licensed version.

## Download

Download the licensed version from [your account](https://orgchartjs.balkan.app/).

## Install

Replace the trial file/s with the licensed one.

## Test

Clear the browser cache (Ctrl + F5) if needed and test with this:

```javascript
console.log(OrgChart.isTrial())
```

> ⚠️ **Warning!**
> If you are using any old framework version, you may need
> [this](https://balkan.app/OrgChartJS/Docs/react) installation guide.
